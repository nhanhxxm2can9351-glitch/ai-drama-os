# AI Drama OS - AI短剧生产操作系统 产品需求文档

## 1. Product Overview
AI Drama OS 是一个从创意到发布的全链路自动化短剧生产平台，通过多 Agent 协作、工作流引擎和可视化无限画布，实现百万级剧集的工业化生产。

- 核心价值：解决角色一致性、画风统一、长剧集生产等行业痛点，将短剧生产效率提升 100 倍
- 目标用户：个人创作者、MCN 机构、影视公司

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| 个人创作者 | 邮箱/手机号注册 | 使用基础功能，创建个人项目 |
| 专业版用户 | 付费订阅 | 高级功能、更多积分、角色市场交易 |
| 企业用户 | 定制化部署 | 资产管理、团队协作、版权管理 |

### 2.2 Feature Module
1. **封面页**: 项目展示、版本信息、设计团队
2. **项目总览**: 产品愿景、9阶段路线图总览
3. **产品路线图**: 从 MVP 到 V3.0 的 9 个阶段详细拆解
4. **核心数据协议**: 统一数据协议、统一 Prompt 协议
5. **系统模块架构**: 7 大核心系统模块关系图
6. **工作流引擎**: 全链路生产工作流、节点定义
7. **多 Agent 团队**: 6 个 Agent 的职责与协作流程
8. **无限画布**: ReactFlow 节点设计、拖拽交互逻辑
9. **商业化模块**: 4 大商业化模式与收费体系
10. **技术开发计划**: 优先开发项、技术栈、里程碑

| Page Name | Module Name | Feature description |

## 3. Core Process
用户通过导航栏切换 10 个页面，每个页面展示相应的设计内容。无限画布页面支持节点拖拽交互演示，代码块支持一键复制。

```mermaid
flowchart LR
    A[封面页] --> B[项目总览]
    B --> C[产品路线图]
    C --> D[核心数据协议]
    D --> E[系统模块架构]
    E --> F[工作流引擎]
    F --> G[多Agent团队]
    G --> H[无限画布]
    H --> I[商业化模块]
    I --> J[技术开发计划]
```

## 4. User Interface Design
### 4.1 Design Style
- 主色：科技蓝 #165DFF
- 辅助色：紫色 #722ED1、橙色 #FF7D00、绿色 #00B42A、红色 #F53F3F
- 中性色：深灰 #1D2129、中灰 #4E5969、浅灰 #C9CDD4、背景 #F2F3F5
- 字体：Inter + JetBrains Mono（代码）
- 按钮风格：圆角 8px，悬停阴影
- 布局：卡片式布局，12列网格，1440px 最大宽度
- 动效：平滑过渡 0.3s，悬停缩放，滚动渐入

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 封面页 | Hero区域 | 大标题、渐变背景、粒子动效、居中布局 |
| 项目总览 | 双栏布局 | 左侧愿景卡片，右侧横向时间轴 |
| 产品路线图 | 网格卡片 | 3x3 卡片网格，悬停上浮效果 |
| 核心数据协议 | 代码块 | 深色背景、语法高亮、复制按钮 |
| 系统模块架构 | 分层图 | 从上到下四层，连接线动画 |
| 工作流引擎 | 流程图 | 横向节点连接，可点击高亮 |
| 多Agent团队 | 角色卡片 | 头像、职责标签、协作箭头 |
| 无限画布 | 三栏布局 | 左窄中宽右窄，拖拽交互演示 |
| 商业化模块 | 定价表 | 三列定价卡片，突出推荐方案 |
| 技术开发计划 | 看板布局 | 三列任务看板，拖拽排序演示 |

### 4.3 Responsiveness
- Desktop-first（1440px）
- Tablet（1024px）：两栏变一栏，卡片自适应
- Mobile（768px）：导航折叠、单列布局、触控优化

### 4.4 Visual Effects
- 背景：深色渐变 + 网格纹理 + 光晕效果
- 卡片：毛玻璃效果、微妙边框、投影
- 图标：Lucide 图标库，统一 24px 尺寸
- 动效：页面切换过渡、滚动触发动画、悬停微交互
